# Partner & Developer Program (Phase 4)

**Status:** Draft  
**Owner:** Strategic Partnerships Manager  
**Last Updated:** 2025-05-01

## 1. Program Levels

### 1.1 Registered Developer (Open)
*   **Target:** Independent devs, niche marketplaces.
*   **Access:** Verified Valuation API (Rate limited: 100/day).
*   **Requirements:** Email verification, Terms of Service.
*   **Cost:** Free.

### 1.2 Certified Partner (Vetted)
*   **Target:** Regional restoration firms, boutique insurers.
*   **Access:**
    *   Unlimited Valuation API.
    *   Data Sync API (Read-only, with user consent).
    *   "MyARK Certified" Badge for their marketing.
*   **Requirements:** Business verification, 10+ active shared users.
*   **Cost:** $499/month or Revenue Share.

### 1.3 Premier Enterprise (Strategic)
*   **Target:** National Carriers (State Farm, Allstate), Big Tech.
*   **Access:**
    *   Full White-Label Widget SDK.
    *   Real-time "Firehose" analytics (anonymized).
    *   Dedicated Solution Architect.
*   **Requirements:** Multi-year contract, >10k users.
*   **Cost:** Custom Negotiated.

## 2. Onboarding Flow
1.  **Application:** Web form + API Key generation (Sandbox only).
2.  **Sandbox Testing:** Mock data environment.
3.  **Certification:** Automated check of API usage patterns + compliance review.
4.  **Production Keys:** Released upon contract signature (for Certified/Premier).

## 3. Developer Portal (Planned)
*   Dashboard showing API usage, error rates, and active connected users.
*   Docs hosted at `developer.myark.io` (Phase 4.1).
