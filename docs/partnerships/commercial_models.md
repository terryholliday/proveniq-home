# Commercial Models (Enterprise Phase 4)

## 1. SaaS Subscription (Per Seat)
*   **Target:** Restoration Vendors, Professional Organizers.
*   **Pricing:** $49/agent/month.
*   **Includes:** Unlimited capture, PDF reports, Admin Dashboard.

## 2. API Usage (Utility)
*   **Target:** Marketplaces, Insurtech.
*   **Pricing:**
    *   **Valuation:** $0.50 per item.
    *   **Verification (Visual Truth):** $2.00 per item.
*   **Commit:** Minimum $1k/month.

## 3. Revenue Share (B2B2C)
*   **Target:** Insurance Carriers.
*   **Model:** Free for Policyholders.
*   **Rev Share:** MyARK pays Carrier 10% of any "Monetization Event" (e.g., user sells an item via MyARK).
*   **Rationale:** Reduces Carrier friction to bundle us; we monetize the *asset lifecycle*.
